# vue-google-maps

A Vue 3 & Google Maps Integration Demo.

## Project Setup
```sh
npm install
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Compile and Minify for Production
```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)
```sh
npm run lint
```

### Clean development and production folders
```sh
npm run clean
```
